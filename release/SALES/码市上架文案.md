### 项目简介

这是一套**储能电站监控系统（忆小排）**完整源码，基于 React 19 + TypeScript + ECharts + Tailwind CSS 构建，实现储能电站母排-电池协同监控联动，适用于工业监控、能源管理、智慧园区等场景。

**技术成熟，代码规范，可直接商用或二次开发！**

### 核心功能

| 模块 | 功能说明 |
|------|----------|
| 📊 实时监控仪表盘 | 温度场可视化、设备状态、系统效率、告警统计 |
| 🔋 电池状态监测 | 电压、电流、SOC、SOH 实时数据展示 |
| 🌡️ 温度趋势分析 | 历史温度数据图表，支持时间筛选 |
| ⚠️ 智能告警系统 | 多级告警分级、历史查询、告警处理 |
| 🔧 设备管理 | 设备列表、状态监控、故障诊断、分组管理 |
| 📈 数据分析 | 能耗分析、效率统计、趋势预测 |
| 📝 报表管理 | 自动生成运维报表，支持导出 |
| 👤 用户权限管理 | 基于角色的权限控制（管理员/运维/工程师/访客） |
