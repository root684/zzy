# 储能电站监控系统（忆小排）

## 系统简介

忆小排是基于忆阻存算一体技术的储能电站母排-电池协同监控联动系统，实现母排-电池的全维度感知、毫秒级预警、智能化联动、闭环式调控，打造从"被动抢修"到"主动预防"的储能安全防护体系。

## 技术栈

- **前端**: React + TypeScript + Tailwind CSS + ECharts
- **构建工具**: Vite
- **路由**: React Router

## 系统功能

### 1. 用户认证与授权
- 登录/注册功能
- 基于角色的权限管理
- 用户信息管理

### 2. 实时监控
- 母排-电池温度场可视化
- 运行参数仪表盘
- 异常状态实时监测
- 区域状态分布展示

### 3. 历史数据管理
- 历史数据查询与筛选
- 温度趋势分析图表
- 数据导出功能
- 分页展示

### 4. 告警管理
- 告警信息展示与分级
- 告警历史查询
- 告警处理功能
- 告警统计图表

### 5. 设备管理
- 设备列表与状态展示
- 设备配置管理
- 设备故障诊断
- 设备分组与搜索

### 6. 系统设置
- 系统参数配置
- 用户权限管理
- 告警阈值设置
- 配置导入导出

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 系统特点

1. **美观直观的界面**: 采用现代化的工业监控系统设计，深色主题，响应式布局
2. **实时数据监测**: 毫秒级数据刷新，实时展示母排-电池温度场
3. **智能告警系统**: 多级告警机制，及时发现和处理异常
4. **丰富的数据可视化**: 使用 ECharts 实现各种图表展示
5. **完善的设备管理**: 设备状态监控、配置管理、故障诊断
6. **灵活的系统设置**: 支持系统参数配置、用户权限管理、告警阈值设置
7. **安全可靠**: 基于角色的权限控制，确保系统安全

## 注意事项

1. 系统运行需要后端 API 提供数据支持
2. 建议使用 Chrome 或 Firefox 浏览器访问
3. 系统支持响应式布局，可在不同屏幕尺寸下使用
4. 首次登录后请修改默认密码
